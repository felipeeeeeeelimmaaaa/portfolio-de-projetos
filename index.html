☁️ 4. Publicando no GitHub

  1. Crie um repositório com o nome portfolio-de-projetos.

  2. Faça upload dos arquivos no repositório (index.html, style.css, pasta img).

  3. Vá em Settings > Pages.

  4. Em Source, selecione main e /root.

  5. Clique em Save — o link será gerado (algo como https://seunome.github.io/portfolio-de-projetos/).

3. Ative o GitHub Pages

  • Vá para a aba Settings do repositório.

  • Clique em "Pages" no menu lateral.

  • Em "Source", escolha:

    • Branch: main

    • Pasta: / (root)

  • Clique em "Save"
